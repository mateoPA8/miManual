CSS

Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) o CSS es el lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML (en-US) (incluyendo varios languages basados en XML como SVG, MathML o XHTML). CSS describe como debe ser renderizado el elemento estructurado en la pantalla, en papel, en el habla o en otros medios.

Nombre de Color Podemos especificar un color usando un nombre predefinido. p{color: red;}

ELEMENTO DESCRIPCION EJEMPLO
Selector Universal (*) Selecciona todos los elementos. *{color: green;}
Selector de Tipo(p) Permite seleccionar elementos usando el nombre del elemento. p{color: red;}Seleccionamos todos los elementos p del documento.
SELECTOR DE CLASE (.foo) Permite seleccionar elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector. p class=”foo bar”> ¡HOLA MUNDO! p class=”foo baz”> ¡ADIÓS MUNDO! .foo{color: blue;}
SELECTORES DE ID (#foo) Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector. p id=”foo”> ¡HOLA MUNDO! /p #foo{color: blue;}
SELECTORES DE ATRIBUTO (a[foo]) Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. a[title]{color: blue;}Seleccionamos los elementos a, que tengan el atributo title.
PSEUDO-CLASSES Permite seleccionar elementos en base a su estado (:disabled), posicion de los elementos en el documento(:nth-child) e interacción del usuario(:hover) selector:pseudo-class { … } button:hover{color: blue;}
PSEUDO- ELEMENTOS Seleccionan parte de un elemento en lugar del elemento mismo. Los pseudo-elementos se añaden a los selectores y permiten añadir estilos a una parte concreta de los elementos seleccionados. selector::pseudo-elemento { … } p::first-line{color: blue;}
UNION DE SELECTORES Selecciona la unión de los elementos seleccionados por cada selector de la lista Selector-1, Selector-2, … { }
SELECCIONAR ELEMENTOS DESCENDIENTES Seleccionamos todos los elementos seleccionados por B que son descendientes de los elementos seleccionados por A A B {}
SELECCIONAR ELEMENTOS HIJOS Seleccionamos todos los elementos seleccionados por B que son hijos de los elementos seleccionados por A A>B{}
SELECCIONAR ELEMENTOS HERMANOS Seleccionamos todos los elementos seleccionados por B que son hermanos de los elementos seleccionados por A A~B {}
SELECCIONAR ELEMENTOS HERMANOS ADYACENTES Seleccionamos todos los elementos seleccionados por B que son hermanos y que están inmediatamente despues de los elementos seleccionados por A. A+B {}

MODELO DE CAJAS

Cada elemento de un documento se considera una caja. CSS determina el tamaño, posicion y propiedades(color, fondo, el ancho del borde, etc).

Cada caja está compuesta de cuatra partes:Contenido, Relleno, Borde, Margen.

AREA DESCRIPCION PROPIEDADES
CONTENIDO Es el área donde se encuentra el contenido del elemento(texto, imagen, etc). width,min-width,max-width,height,min-height,max-height
RELLENO Es el área de relleno del elemento. padding-top,padding-right,padding-bottom,padding-left
BORDE Es el borde del elemento. border-width,border-style,border-color
MARGEN Es el margen del elemento. Se usa para separar el elemento de sus vecinos. margin-top,margin-right,margin-bottom,margin-left

FLEXBOX

La propiedad display con valor flex o inline-flex nos permite establecer un contenedor flexible. Los hijos del contenedor flexible se llaman elementos flexibles.

EJE PRINCIPAL

Es un eje a lo largo del cual se disponen los elementos flexibles. No es necesariamente horizontal; depende de la propiedad flex-direction

Lado inicial del eje principal | Lado final del eje principal

Los elementos flexibles se colocan dentro del contenedor comenzando desde el Lado inicial del eje principal yendo al lado final del eje principal

Lineas flexibles

Las líneas flexibles son construcciones imaginarias que alojan a los elementos flexibles. Pueden existir varias o una linea flexible dentro de un contenedor flexible0

Tamaño principal

El tamaño principal de un contenedor o un elemento flexible es su ancho (width) cuando el eje principal es horizontal y es su alto (height) cuando el eje principal es vertical.

EJE TRANSVERSAL

Es un eje a lo largo del cual se disponen las lineas flexibles. Es un eje perpendicular al eje principal.

Lado inicial del eje transversal | Lado final del eje transversal

Las líneas flexibles se colocan en el contenedor comenzando en el lado inicial del eje transversal yendo hacia el lado final del eje transversal.

Tamaño transversal

El tamaño transversal de un contenedor o un elemento flexible es su alto (height) cuando el eje transversal es vertical y es su ancho (width) cuando el eje transversal es horizontal.
